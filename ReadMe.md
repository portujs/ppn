# ppn

> Pesquisar pacotes [Node.js](https://nodejs.org/) em português via linha de comando.

## Instalação

Via linha de comando com [NPM](https://npmjs.com/):

```
$ npm install -g ppn
```

## Uso

```
$ ppn <termo>
```

**Exemplo:**

Será exibido somente pacotes que tenham a descrição em português.
